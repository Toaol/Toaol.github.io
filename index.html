0. 什么是 Octopress

Octopress 是基于 Jekyll 的一个静态博客生成工具，使用 Ruby 语言编写，能通过简单的配置生成一个静态网站，还能托管到 Github Pages，免费并且独立。

除此之外，Octopress 还提供了许多的第三方插件和主题，要是动手能力强还可以自己去制作插件。
